# ☕ Esto es Java — Laboratorio del Roadmap

![Java](https://img.shields.io/badge/Java-17+-orange)
![Estado](https://img.shields.io/badge/estado-en%20evolución-blue)
![Labs](https://img.shields.io/badge/laboratorios-experimentales-green)

Este repositorio forma parte del proyecto **Esto es Java**.

No es simplemente un repositorio de ejemplos.

Es un **laboratorio viviente** donde se experimenta con los conceptos del roadmap de aprendizaje del proyecto.

La idea es simple:

> Aprender Java no es memorizar sintaxis.  
> Es **entender cómo funciona el lenguaje y cómo se ejecuta el código**.

Aquí cada módulo del roadmap se transforma en un espacio de experimentación donde exploramos cómo funciona realmente Java.

---

## 📘 Sobre el proyecto "Esto es Java"

"Esto es Java" es una guía diseñada para aprender el lenguaje **desde su funcionamiento interno**, desarrollando modelos mentales sólidos y criterio técnico real.

El proyecto combina:

- explicación conceptual
- experimentación práctica
- análisis del comportamiento de la JVM
- evolución del lenguaje Java

Más información del proyecto:

👉 https://scalamentis.vercel.app/esto-es-java

---

## 🧭 Cómo funciona este repositorio

Este repositorio acompaña el **roadmap de aprendizaje** del proyecto.

Cada módulo del roadmap se desarrolla dentro de su **propia rama**.

Por ejemplo:

```
main
│
├── modulo1
├── modulo2
├── modulo3
├── modulo4
└── ...
```

Cada rama contiene:

- laboratorios prácticos
- experimentos en código
- documentación técnica
- exploraciones del lenguaje

Esto permite que cada etapa del aprendizaje tenga **su propio espacio de experimentación**.

---

## 🌱 Un laboratorio en evolución

Este repositorio **no está pensado como algo estático**.

A medida que el roadmap evoluciona, también lo hacen:

- los laboratorios
- los ejemplos de código
- las explicaciones
- los experimentos con la JVM

Por eso este repositorio funciona como un **registro vivo del aprendizaje**.

Cada módulo agrega una nueva capa de comprensión sobre Java.

---

## 🔬 Qué encontrarás aquí

Dependiendo del módulo, los laboratorios pueden explorar temas como:

- funcionamiento interno de Java
- cómo compila Java a bytecode
- comportamiento de la JVM
- gestión de memoria
- classloaders
- string pool
- evolución del lenguaje
- diferencias entre Java legacy y moderno
- refactorización de código
- experimentos de rendimiento

El objetivo no es solo ejecutar ejemplos, sino **observar cómo se comporta realmente el lenguaje**.

## 🧪 Cómo usar este repositorio

### 1️⃣ Clonar el repositorio

```bash
git clone https://github.com/JuliaDaniR/esto-es-java.git
```
---
### 2️⃣ Cambiar a la rama del módulo que quieras explorar

Ejemplo:

```bash
git checkout modulo1
```
Cada módulo vive en su propia rama.

---
### 3️⃣ Explorar los laboratorios

Dentro de cada módulo encontrarás dos carpetas principales:
```
laboratorio/
labs/
```
--- 

| Carpeta | Contenido |
|--------|-----------|
| `laboratorio` | explicación conceptual |
| `labs` | experimentos en código |

La idea es **leer la explicación y luego ejecutar el laboratorio**.
